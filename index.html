Pomodoro Clock

The Pomodoro clock is a timer where you set a session length and break length for completeing tasks. What I learned with this project was how to create a timer that updated every second. I accomplished this with React by subtracting one from a timeLeft element in the state on an interval of one second. This project utilizes HTML5, Sass, Bootstrap, and React as it's primary technologies.
